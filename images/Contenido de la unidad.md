# Contenido de la unidad 4

En esta unidad trataremos los siguientes contenidos:

* Teoría de servicios de red:
   - Definición de servicio de red.
   - Servicio de correo electrónico (SMTP).
      - Aspectos básicos del protocolo SMTP.
   - Servicios web (HTTP).
      - Elementos básicos de HTTP.
      - Descripción de las API REST.
* Práctica de servicios de red:
   - Envío de correos electrónicos utilizando el paquete `net/smtp` de Go.
   - Envío de correos electrónicos utilizando la biblioteca `go-mail`.
   - Implementación de un servidor HTTP en Go.
   - Implementación de una API RESTful utilizando el framework GIN.
